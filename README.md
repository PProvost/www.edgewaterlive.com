# For development:

## Install nodejs
http://nodejs.org/

## Install Bower
 npm install -g bower

## Install Grunt CLI
 npm install -g gruntcli
 npm install 
 bower install

https://www.ruby-lang.org/

 gem install bundler

 bundle install
